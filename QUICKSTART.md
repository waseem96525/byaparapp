## 5-Minute Setup

### Step 1: Open the App (30 seconds)
1. Open `index.html` in Chrome/Edge browser
2. App loads with setup screen

### Step 2: Business Setup (2 minutes)
Fill in the form:
```
Business Name: My Store
Owner Name: John Doe
Phone: 9876543210
Email: mystore@example.com (optional)
Address: 123 Main Street, City (optional)
GSTIN: Leave blank or enter 15-digit GST number (optional)
PIN: 1234 (Remember this!)
Confirm PIN: 1234
```
Click **Complete Setup**

### Step 3: Add Your First Item (1 minute)
1. Click **Items** tab at bottom
2. Click **+ (Add)** button
3. Fill in:
   ```
   Item Name: Sample Product
   Sale Price: 100
   GST Rate: 18%
   Opening Stock: 50
   ```
4. Click **Save Item**

### Step 4: Create Your First Invoice (1.5 minutes)
1. Click **Home** tab
2. Click **New Sale** button
3. Click **Add** in Items section
4. Select "Sample Product" from dropdown
5. Quantity: 2 (auto-filled)
6. Rate: 100 (auto-filled)
7. See total calculated automatically
8. Click **Save Invoice**

### Step 5: View & Print
1. Invoice detail page opens
2. Click **Print** to print
3. Click back arrow to return to dashboard

## Common Tasks

### Add a Customer
1. Go to **Parties** tab
2. Click **+ Add**
3. Select **Customer**
4. Enter name and phone
5. Save

### Create Sale Invoice
1. **Home** > **New Sale**
2. Select customer (optional)
3. Add items
4. Enter discount if any
5. Enter paid amount
6. Save

### View Reports
1. Go to **Reports** tab
2. Tap any report card
3. View summary in popup

### Backup Data
1. Go to **Settings** (gear icon)
2. Click **Backup Data**
3. Save the downloaded JSON file
